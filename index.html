معرفی پایتون

پایتون یک زبان برنامه نویسی سطح بالا است که یادگیری آن آسان است و به طور گسترده برای توسعه وب، علم داده، اتوماسیون و غیره استفاده می شود. به دلیل نحو ساده و خوانا معروف است.

برای: توسعه وب، توسعه نرم افزار موبایل و کامپیوتر، هوش مصنوعی، کار با داده ها... استفاده میشود

نصب پایتون

قبل از شروع این دوره آموزشی پایتون ابتدا باید پایتون را روی کامپیوتر خود نصب کنید. برای نصب پایتون بر روی کامپیوتر خود، مراحل زیر را دنبال کنید: دانلود پایتون: به وب سایت رسمی پایتون در بروید. در صفحه اصلی، بخش "دانلودها" را مشاهده خواهید کرد. بر روی دکمه "دانلود پایتون" کلیک کنید.

شروع اموزش پایتون

Variables are used to store data values. In Python, you do not need to declare the type of a variable; Python automatically determines it based on the value assigned.

x = 5
y = "Hello, World!"
print(x)
print(y)

Here, x is assigned an integer value and y is assigned a string. The print() function is used to display the values.

Functions in Python

Functions in Python allow you to group code into reusable blocks. You can define a function using the def keyword.

def greet(name):
   print("Hello, " + name)

greet("Alice")

This function greet() takes one parameter and prints a greeting message using the passed argument.

Loops in Python

Loops allow you to execute a block of code multiple times. Python supports two types of loops: for and while.

for i in range(5):
   print(i)

The for loop above prints numbers from 0 to 4. The range(5) function generates a sequence of numbers from 0 to 4.
